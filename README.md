# CeliAPP

[Link do vídeo mostrando protótipo FIGMA](https://youtu.be/W5M6cFiKQZY)
[Link do excalidraw com ideias iniciais](https://excalidraw.com/#room=cab5d65cbb91e1c52e05,QWHnkkuUt3VLCcBhOlFiPw)

Quero fazer um projeto real para aprender um pouco do processo inteiro de desenvolvimento, desde levantamento de requisitos e personas, versionamento de código com GIT, criação de esteira CI/CD e práticas DevOPS, desenvolvimento de um protótipo com FIGMA e testes de usabilidade, estudo da arquitetura do sistema, desenvolvimento do sistema back end com Java+SpringBoot, desenvolvimento do front end com React, integração com banco PostgreeSQL, uso de uma API externa, documentação com swagger, testes unitários, um pouco de docker e kubernetes e posteriormente deploy com AWS.
